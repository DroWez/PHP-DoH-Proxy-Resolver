# 🚀 PHP DoH Proxy Resolver

Лёгкий, быстрый и гибкий **DNS-over-HTTPS (DoH) прокси-резолвер на PHP**.

Работает на обычном **shared-хостинге** без Docker и без root-доступа.

---

## 🚀 Возможности

- Приём DoH‑запросов (GET/POST)
- Блокировка рекламы (списки AdGuard)
- Белый/чёрный список IP
- Upstream DNS + маршрутизация по доменам
- Принудительные ответы (override)
- Кеширование (APCu — опционально)
- Логи в SQLite
- Веб‑панель: статистика, живые запросы, журнал, топы, настройки

---

## 🔧 Требования

- PHP 7.4+
- SQLite3
- cURL
- APCu (по желанию)

## 🔐 Как установить пароль администратора

Пароль хранится в файле **admin.php** в виде bcrypt‑хеша:
```php
$USERNAME = "admin";
$PASSWORD_HASH = '$2y$12$...';
```
Чтобы задать свой пароль:
Выполните в PHP (через консоль, phpinfo, отдельный файл — как удобно):
```php
<?php
echo password_hash("МОЙ_НОВЫЙ_ПАРОЛЬ", PASSWORD_DEFAULT);
```
Скопируйте полученный хеш.
Вставьте его в admin.php:
```php
$PASSWORD_HASH = 'скопированный_хеш';
```
Готово — новый пароль активен.

## ⚙️ Настройка

Настройки DNS выполняются через веб-интерфейс.

Параметры:

Анонимный режим — отключить логирование

Белый список IP — разрешенные IP (по одному на строку)

Черный список IP — заблокированные IP (по одному на строку)

Upstream DNS — серверы проксирования (URL на строку)

Фильтры рекламы — списки блокировки (URL на строку)

Маршрутизация — upstream для доменов (домен=upstream)

Переопределение — принудительный IP (домен=IP)

## ⚠️ Возможные проблемы

### Не записываются логи

Проверьте:

* права на запись в директорию
* отключен ли анонимный режим

### Не загружаются фильтры

Проверьте:

* доступность URL списков
* удалите файл **filters.cache** для принудительной загрузки

### Медленные DNS запросы

Рекомендуется:

* включить APCu
* проверить доступность upstream DNS
